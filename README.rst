PyLife - a Game of Life implementation

by Jacob Paulette
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Implemented in Python 3.
Reads rle pattern files.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~
External Dependencies:

scipy
numpy
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Usage:

adjust SETTINGS section in main.py to taste.

in terminal
>python main.py    # will use default pattern rle.string, in rle.py
>python main.py rle_file    # can read any file, as long as it's in rle format
